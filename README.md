## 🎀 아이디어 필요성  
- 임산부를 위한 앱 서비스는 있으나, 산후 우울증과 불안을 예방하고 임산부와 가족 간 유대감을 강화하는 서비스는 부족한 실정이다. 임산부의 스트레스와 우울이 태아에 미치는 영향에 대해 이미 여러 국내외 학술지와 전문가의 의견이 존재한 만큼 저출산·고령화 사회에서 산후 우울과 불안을 예방하고, 가족 간 유대감을 강화하는 디지털 서비스가 필요한 것을 알 수 있다.
